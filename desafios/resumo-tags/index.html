Formulário (início)

No módulo sobre formulário aprendemos sobre os principais inputs utilizados.

  • Text: campo de texto com uma linha
  • Number: campo que aceita apenas números
  • Color: seletor de cores
  • Email: campo validado com um endereço de email
  • Url: campo que aceita apenas links (ex: https://www.dio.me/)
  • Date: apresenta uma interface para seleção de data (o usuário pode digitar a data, se preferir)
  • Checkbox: caixa de marcação (o usuário pode selecionar várias)
  • Radio: caixa de marcação (não pode selecionar mais de um com o mesmo name)
  • File: permite que o usuário selecione um arquivo

Por fim, utilizamos os métodos GET e POST para compreender suas características:

  • GET: os dados do formulário são enviados na URL (fica visível para o usuário)
  • POST: os dados são enviados na requisição HTTP (não aparece na URL como o GET)

Tabela (início)

Para criar uma tabela em HTML é necessário iniciar utilizando a tag table.
Cada linha pode ser inserida utilizando tr (table row) e as colunas são inseridas com td (table data). Já para o cabeçalho da tabela utilizamos o th.


                <table>
                    <tr>
                        <td>Linha 1, Coluna 1</td> <td>Linha 1, Coluna 2</td>
                    </tr>
                    <tr>
                        <td>Linha 2, Coluna 1</td> <td>Linha 2, Coluna 2</td>
                    </tr>
                </table>

O atributo summary é utilizado para descrever a tabela (não aparece para o usuário) e o caption é o título da tabela (aparece para o usuário).
Também podemos utilizar o cellspacing para definir o espaço entre as célular e o cellpadding para o espaço entre o conteúdo e a borda da célula.

Mídia (início)

As mídias são elementos que podem ser inseridos nas páginas a partir das tags img, audio e video.
Nelas é possível ajustar o tamanho da mídia (width e height), incluir controles de mídia com o controls e adicionar uma descrição para leitores de tela com o alt.

O title define um balão que aparece ao passar o mouse por cima da imagem e a tag track é utilizada para inserir legendas.

Semântica (início)

A ideia principal deste módulo é compreendermos a importância de criar sites que todas as pessoas possam usar, inclusive aquelas com deficiências visuais, motoras, auditivas ou cognitivas.
Abaixo temos um resumo dos tópicos abordados.

  • web scraping: técnica de coletar dados de uma página da web
  • header: cabeçalho de uma página ou seção (logo, título, menu...)
  • main: não pode ser filho de outros elementos e não pode haver mais de um elemento main no mesmo documento
  • footer: rodapé da página ou seção (contatos, links úteis...)
  • section: seção genérica do conteúdo (tópicos relacionados)
  • aside: conteúdo relacionado ao principal mas não essencial (barras laterais, links extras...
  • nav: links de navegação da página (menu, sumário...)
  • article: conteúdo independe e reutilizável (notícia, post, comentário)
  • blockquote: bloco de citação longa
  • q: citação curta dentro do texto
